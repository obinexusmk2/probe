MMUKO OS — Network Probing Protocol

Ask the
right question.

PROBE is a three-state consensual network questioning system.
WHO. WHAT. WHEN. WHERE. WHY. HOW.
Returns YES, NO, or MAYBE.

// Try it
Enter an address and press PROBE...
  probe.exe
$ .\probe.exe 8.8.8.8
[YES] WHO: 8.8.8.8 → dns.google
[YES] WHAT: open ports [443]
[NO] WHEN: last_seen=1773099653
[YES] WHERE: 8.8.8.8 reachable=YES

$ .\probe.exe 1.1.1.1 WHAT
[YES] WHAT: open ports [80 443 8080]

$ .\probe.exe network 192.168.1.0/24
[MAYBE] WHERE: 192.168.1.1 reachable=MAYBE
[YES] WHERE: 192.168.1.5 reachable=YES
